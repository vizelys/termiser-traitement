# TERMISER Traitement — Landing Page

Landing page premium pour **TERMISER Traitement**, artisan-expert du bois en Gironde depuis 1991.

**Live** → [vizelys.github.io/termiser-traitement](https://vizelys.github.io/termiser-traitement/)

---

## Fichiers

| Fichier | Usage |
|---|---|
| `index.html` | Page principale — à déployer sur n'importe quel hébergeur |
| `termiser-standalone.html` | Version autonome, 100 % hors-ligne (polices et images embarquées) |
| `images/` | Photos des interventions (charpente, mérule, termites, humidité, artisan) |

## Stack

HTML / CSS / JS pur — aucun framework, aucun build step.

## Déploiement

**GitHub Pages** (actuel) — chaque `git push` sur `main` met à jour le site automatiquement.

**Autres options :**
- Netlify : glisser-déposer `index.html` + `images/` dans l'interface
- Hébergement mutualisé : FTP des fichiers à la racine `public_html/`
